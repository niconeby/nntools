Kebijakan Privasi

Kebijakan privasi MediaAgni Tools terkait penggunaan layanan dan perlindungan data.

Terakhir diperbarui: 27 Januari 2026

MediaAgni Tools menghargai dan melindungi privasi setiap pengunjung. Dokumen ini menjelaskan bagaimana kami mengelola informasi pengguna saat menggunakan layanan kami.

1. Informasi yang Kami Kumpulkan

Kami tidak mengumpulkan, menyimpan, atau merekam password yang dimasukkan pengguna pada halaman Cek Keamanan Password.

  • Diproses secara lokal di perangkat pengguna (client-side)
  • Tidak disimpan dalam database apa pun
  • Tidak dikirim ke server kami dalam bentuk teks asli

2. Penggunaan Layanan Pihak Ketiga

Website ini menggunakan layanan pihak ketiga Have I Been Pwned (HIBP) â€“ Pwned Passwords API untuk memeriksa apakah password pernah terlibat dalam kebocoran data.

  • Password di-hash (SHA-1) di browser pengguna
  • Hanya sebagian hash yang dikirim ke server HIBP
  • Password asli tidak pernah diketahui oleh kami

3. Informasi Log

Kami dapat mengumpulkan informasi non-pribadi secara otomatis, seperti:

  • Alamat IP
  • Jenis browser
  • Penyedia layanan internet (ISP)
  • Tanggal dan waktu akses
  • Halaman yang dikunjungi

4. Cookie dan Web Beacon

MediaAgni Tools menggunakan cookie untuk menyimpan preferensi pengguna dan meningkatkan pengalaman penggunaan website.

5. Mitra Periklanan

Beberapa mitra periklanan dapat menggunakan cookie atau web beacon. Kami tidak memiliki kendali atas teknologi tersebut.

6. Kebijakan Privasi Pihak Ketiga

Kebijakan ini tidak berlaku untuk situs pihak ketiga. Kami menyarankan pengguna membaca kebijakan privasi masing-masing pihak.

7. Persetujuan

Dengan menggunakan layanan MediaAgni Tools, Anda menyetujui kebijakan privasi ini.
