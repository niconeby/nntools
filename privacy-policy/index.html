Kebijakan Privasi – MediaAgni Tools

Terakhir diperbarui: 27 Januari 2026

MediaAgni Tools menghargai dan melindungi privasi setiap pengunjung. Dokumen Kebijakan Privasi ini menjelaskan bagaimana kami mengelola informasi pengguna saat mengakses dan menggunakan layanan kami.

1. Informasi yang Kami Kumpulkan

Kami tidak mengumpulkan, menyimpan, atau merekam password yang dimasukkan pengguna pada halaman Cek Keamanan Password.

  • Diproses secara lokal di perangkat pengguna (client-side)
  • Tidak disimpan dalam database apa pun
  • Tidak dikirim ke server kami dalam bentuk teks asli (plain text)

2. Penggunaan Layanan Pihak Ketiga

Website ini menggunakan layanan pihak ketiga Have I Been Pwned (HIBP) – Pwned Passwords API untuk memeriksa apakah sebuah password pernah terlibat dalam kebocoran data.

  • Password diubah menjadi hash (SHA-1) di browser pengguna
  • Hanya sebagian kecil hash yang dikirim ke server HIBP
  • Password asli tidak pernah diketahui atau disimpan oleh kami

3. Informasi Log

Seperti kebanyakan situs web, MediaAgni Tools dapat mengumpulkan informasi non-pribadi secara otomatis, seperti:

  • Alamat IP
  • Jenis browser
  • Penyedia layanan internet (ISP)
  • Tanggal dan waktu akses
  • Halaman yang dikunjungi

Informasi ini digunakan untuk analisis, pengelolaan situs, dan peningkatan kualitas layanan.

4. Cookie dan Web Beacon

MediaAgni Tools menggunakan cookie untuk menyimpan preferensi pengunjung dan mengoptimalkan pengalaman pengguna.

Cookie Google DoubleClick

Google, sebagai vendor pihak ketiga, menggunakan cookie untuk menampilkan iklan di website ini. Cookie DoubleClick memungkinkan Google menampilkan iklan yang relevan berdasarkan kunjungan pengguna.

Pengguna dapat menonaktifkan cookie iklan Google melalui kebijakan privasi Google.

5. Mitra Periklanan

Beberapa pengiklan di website kami dapat menggunakan cookie dan web beacon. MediaAgni Tools tidak memiliki kendali atas cookie yang digunakan oleh pengiklan pihak ketiga.

6. Kebijakan Privasi Pihak Ketiga

Kebijakan Privasi MediaAgni Tools tidak berlaku untuk situs atau pengiklan lain. Kami menyarankan pengguna untuk membaca Kebijakan Privasi masing-masing pihak ketiga.

7. Persetujuan

Dengan menggunakan website kami, Anda menyetujui Kebijakan Privasi ini dan seluruh ketentuan yang berlaku.

8. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai Kebijakan Privasi ini, silakan hubungi kami melalui halaman Contact Us.
