Catmint

Catmint adalah utilitas CLI untuk membuat dan memverifikasi hash file atau folder, membantu memastikan integritas file secara cepat dan konsisten.

catmint
catmint hash -f porto.html -a md5
MD5 hash of file porto.html: 7e12ac023f78bfd35c5d7a70a93b86ab
catmint
catmint verify -f porto.html -a md5 -hash 7e12ac023f78bfd35c5d7a70a93b86ab
File porto.html: hash matches!
